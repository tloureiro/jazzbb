# Keyboard Shortcuts

_Agents: whenever a shortcut definition, label, binding, or purpose changes, update this file accordingly so the macOS and Windows/Linux sections remain accurate before submitting your changes._

## macOS

| Shortcut ID | Description | Keys | Notes |
|-------------|-------------|------|-------|
| `save-note` | Save the current note | `⌘S` | |
| `open-shortcuts` | Open the shortcuts help modal | `⌘/` | |
| `open-command-palette` | Open the command palette | `⌘K` | |
| `toggle-top-bar` | Toggle the header top bar | `⇧⌘H` | |
| `toggle-sidebar` | Toggle the vault sidebar | `⇧⌘B` | Vault mode only |
| `toggle-outline` | Toggle the outline panel | `⇧⌘O` | |
| `open-file` | Open a Markdown file | `⌘⌥O` | |
| `close-document` | Close the current document | `⌘⌥W` | |
| `toggle-grammar-checks` | Toggle grammar checks (browser + Grammarly) | `⌘⌥G` | |
| `toggle-heading-collapse` | Toggle collapse for the current heading | `⌘⌥K` | |
| `toggle-frontmatter-editor` | Toggle the embedded frontmatter editor | `⌘⌥F` | Requires frontmatter |
| `toggle-plain-markdown` | Toggle plain Markdown view | `⌘⌥M` | |
| `delete-line` | Delete the current line | `⌘D` | |
| `escape` | Close dialogs or dismiss overlays | `Esc` | |
| `search-notes` | Open vault-wide search | `⌘P` | Vault mode only |
| `new-note` | Create a new note | `⌘N` | Vault mode only |
